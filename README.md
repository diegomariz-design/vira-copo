# Vira Copo Delivery 🍻

Plataforma de delivery de bebidas — site mobile-first.
Slogan: **Gelada, rápida, na sua casa.**

## Conteúdo
- `index.html` — site completo (single file): portão +18, catálogo, carrinho, checkout com confirmação de maioridade e aviso de documento na entrega.
- `vercel.json` — configuração de hospedagem estática.

## Publicar na Vercel
1. Suba esta pasta para um repositório no GitHub.
2. Em vercel.com → **Add New → Project** → importe o repositório.
3. Framework Preset: **Other** (site estático). Deploy.

## Antes de divulgar
Troque o WhatsApp placeholder em `index.html` (`CONFIG.whatsapp = "5516999999999"`) pelo número real da loja, no formato `55DDDNUMERO`.

> Protótipo de demonstração. Beba com moderação. Venda proibida para menores de 18 anos.
